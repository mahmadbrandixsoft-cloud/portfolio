# Muhammad Ahmad — Portfolio (Static HTML/CSS/JS)

Pure static site — no build step required.

## Run locally
Just open `index.html` in your browser, or serve it:

```bash
# Python
python3 -m http.server 8000

# Node
npx serve .
```

Then visit http://localhost:8000

## Files
- `index.html` — markup
- `styles.css` — all styling + dark/light theme tokens
- `script.js` — theme toggle, mobile menu, data rendering, scroll reveal

## Deploy
Drop the folder on Netlify, Vercel, GitHub Pages, or any static host.
